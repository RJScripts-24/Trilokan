# Frontend - TriLakan

Modern React fintech dashboard with identity verification, app security, and compliance features.

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety  
- **Vite** - Build tool and dev server
- **Framer Motion** - Animations
- **Radix UI** - Accessible components
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

Application runs on http://localhost:3000

### Build
```bash
npm run build
```

## Project Structure

```
src/
 components/          # All React components
 assets/             # Static assets
 styles/             # Global styles
 App.tsx             # Root with authentication
 main.tsx            # Entry point
```

## Features

- Authentication with animated backgrounds
- Identity Verification (KYC)
- App Security Checker  
- Complaint Management
- News Feed
- Wallet Console
- Support Desk
- User Settings

## Design System

Colors: #000000, #1A1A1A backgrounds | #D3AF37 gold accent

## API Integration

Configure backend connection in src/config/api.ts

Environment: Create .env with VITE_API_BASE_URL
